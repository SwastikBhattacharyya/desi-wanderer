# Desi Wanderer

Desi Wanderer is a full-stack travel blogging website built with Next.js and
TailwindCSS.

## Overview

Desi Wanderer is a full-stack travel blogging platform built with Next.js and
TailwindCSS. It features role-based authentication to distinguish between
administrators, authors and regular users. Administrators and Authors can
create, edit, and delete blog posts using a built-in blog editor, while
signed-in users can read posts and leave comments. Administrators can also
manage the users of the application. The application supports full CRUD
functionality for blog management and includes a responsive, mobile-friendly UI.

## Features

- **Blog Editor**: Authors can create, edit, and delete blog posts.
- **Authentication**: Users must sign in to access authoring or commenting
  features.
- **Comment System**: Signed-in users can comment on blog posts.
- **Blog Management**: Full CRUD (Create, Read, Update, Delete) functionality
  for posts.
- **Responsive Design**: Mobile and desktop-friendly layout.
